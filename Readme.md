## Deskripsi
Program PHP sederhana yang menggunakan form input untuk menampilkan:
- Nama  
- Tanggal Lahir  
- Pekerjaan  

Pada praktikum ini, saya membuat sebuah program PHP dasar yang berfungsi untuk menerima input data berupa Nama, Tanggal Lahir, serta Pekerjaan, kemudian program akan:

Menghitung umur pengguna secara otomatis berdasarkan tanggal lahir

Menampilkan pekerjaan yang dipilih

Menentukan besarnya gaji sesuai pekerjaan menggunakan struktur kontrol switch

Menampilkan seluruh hasil pada halaman yang sama setelah form dikirimkan

Praktikum ini bertujuan untuk mempelajari dasar-dasar PHP, terutama cara menggunakan form HTML, memproses data input menggunakan method POST, serta menerapkan logika pemrograman dalam PHP.
